## Лабораторные работы

### Лабораторная работа №1: «Исследование операции свёртки и её применения в обработке изображений»

### Паспорт лабораторной работы:

**Трудоёмкость**: 4 академических часа (аудиторная работа) + самостоятельная работа

**Формируемые компетенции**: 
- MF-5 (Способен применять продвинутые математические методы)
- MF-3 (Способен применять методы оптимизации)

**Проверяемые индикаторы**	
- MF-5.1 (Знает: основы свертки; Умеет: объяснять применение свертки в CNN), 
- MF-5.2 (Умеет: анализировать данные; аргументировать выводы)

**Формат выполнения**: Письменный отчёт с таблицами, схемами и выводами

**Цель работы**: Изучить операцию дискретной свёртки, исследовать влияние различных фильтров на изображение и понять роль свёртки в архитектуре свёрточных нейронных сетей (CNN).

**Задачи работы**:
1. Изучить теоретические основы дискретной свёртки.
2. Проанализировать влияние различных ядер (фильтров) на изображение.
3. Исследовать свойства свёртки на численных примерах.
4. Сделать выводы о применении свёртки в системах ИИ.


## Практические задачи:

**Оценочное средство**: ОС-2 (практические задачи / типовые расчёты)

**Форма контроля**: Текущий (письменное решение + устная защита)

**Роли по КРМ**: ML Engineer (MLE), AI Architect (AIA)

**Вес в итоговой оценке**: 15%
