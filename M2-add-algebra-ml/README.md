# Модуль 2. Дополнительные главы алгебры

## Контрольно-измерительные материалы

| Параметр | Значение |
|---|---|
| Формируемые компетенции | MF-5 (Способен применять продвинутые математические методы для решения сложных задач ИИ и разработки новых алгоритмов) |
| Проверяемые индикаторы | MF-5.1, MF-5.2 |
| Цель КИМ | Оценить уровень сформированности компетенций в области линейной алгебры: теория линейных пространств и операторов, сингулярное разложение (SVD), спектральный анализ, тензоры и их применение в задачах машинного обучения для ролей ML Engineer и AI Architect |
| Формы контроля (ОС) | Текущий: ОС-2 (практические задачи / типовые расчёты), ОС-4 (тесты), Рубежный: ОС-1 (лабораторные работы), Промежуточный: ОС-5 (зачёт)|
| Шкала оценивания | 10-балльная, в соответствии с ФОС |


## Практические задачи:

**Оценочное средство**: ОС-2 (практические задачи / типовые расчёты)

**Форма контроля**: Текущий (письменное решение + устная защита)

**Роли по КРМ**: ML Engineer (MLE), AI Architect (AIA)

**Вес в итоговой оценке**: 15%
