# Рабочая программа дисциплины

Рабочая программа дисциплины «Дополнительные главы алгебры и анализа для ML» предназначена для подготовки обучающихся к применению математического аппарата в задачах анализа данных, машинного обучения и оптимизации моделей в роли
**ML Engineer** или **AI Architect**



## 1. Общие сведения

| Поле | Значение |
|---|---|
| Название дисциплины | Дополнительные главы алгебры и анализа для ML |
| Образовательная программа | Прикладная информатика (профиль): прикладной ИИ |
| Уровень образования | Бакалавриат |
| Объем | 5 зачетных единиц (180 ак. часов) |
| Формат обучения | очный |
| Форма промежуточной аттестации | зачет |

## 2. Цель и задачи

**Цель:** формирование у обучающихся фундаментальные и прикладные знания, умения и навыки, необходимые для математического описания, анализа и реализации алгоритмов машинного обучения. Особое внимание уделяется методам, непосредственно применяемым в ML: градиентным методам, SVD, PCA, оптимизации.

**Задачи:**

1. сформировать представление о математическом аппарате;
2. научить выполнять матричные и векторные преобразования;
3. развить навыки дифференциального анализа функций нескольких переменных 
    и связать их с методами обучения моделей;
4. освоить методы решения задач оптимизации;
5. показать математические основы алгоритмов обучения моделей.

## 3. Пререквизиты

Дисциплина опирается на базовые курсы математического анализа (MF-1 (Б)), линейной алгебры (MF-1 (Б), MF-5 (Б)),  и теории вероятностей и математической статистики (MF-1 (Б), MF-3 (Б)), дискретной математики (MF-5 (Б)).
