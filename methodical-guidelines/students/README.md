# Рекомендации обучающимся

## 1. Лабораторные работы

### 1.1 Рекомендации по выполнению

1. Начните с теории — ознакомьтесь с математическими основами.
2. Разбейте задачу на шаги — реализуйте по частям.
3. Визуализируйте результаты — графики и изображения помогут понять происходящее.
4. Анализируйте результаты — делайте выводы, сравнивайте подходы.
5. Готовьтесь к защите — вы должны уметь объяснить каждую строку кода.

### 1.2 Вопросы для самопроверки к защите

- Почему вы выбрали именно этот подход?
- Как вы проверяли корректность реализации?
- Что происходит при изменении параметров?
- Какой результат вы ожидали и что получили?
- Где этот метод применяется в реальных задачах ML?

## 2. Практические задачи

### 2.1 Рекомендации по выполнению

1. Решайте задачи регулярно — на каждом практическом занятии.
2. Разбирайте ошибки — анализируйте, где допустили неточность.
3. Объясняйте решение — проговорите вслух каждый шаг.
4. Проверяйте себя — подставляйте ответы в исходное условие.

### 2.2 Вопросы для самопроверки к защите

- Что такое [термин]? Дайте определение.
- Как вы нашли [промежуточный результат]?
- Почему вы использовали именно этот метод?
- Как связан результат с ML?
- Что произойдёт, если изменить [параметр]?

## 3. Коллоквиум

### 3.1 Рекомендации по подготовке

1. Изучите конспекты лекций — выделите ключевые определения и теоремы.
2. Проговорите ответы вслух — это помогает запомнить и структурировать.
3. Приводите примеры из ML — это показывает понимание.
4. Составьте связи между темами — например, как SVD связан с PCA.

### 3.2 Вопросы для самопроверки к защите

- Что такое градиент и как он используется в оптимизации?
- В чём суть метода множителей Лагранжа?
- Что такое SVD и чем оно отличается от диагонализации?
- Как свёртка применяется в CNN?
- В чём отличие ньютоновских и квазиньютоновских методов?

## 4. Зачет

### 4.1 Рекомендации по подготовке

1. Изучите все вопросы из перечня (Приложение 1 к ФОС).
2. Составьте краткие конспекты для каждого вопроса.
3. Проговорите ответы вслух.
4. Приводите примеры из ML.

### 4.2 Общие рекомендации:
    
- Повторите ключевые формулы: градиент, матрица Гессе, SVD, собственные значения.
- Вспомните, как эти методы применяются в ML.
- Отдохните перед зачётом — свежая голова важнее зубрёжки.
