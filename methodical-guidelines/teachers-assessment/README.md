# Методические указания преподавателям по оцениванию

Для всех оценочных средств применяется единая 10-балльная шкала. 
Ключевые опорные точки шкалы соответствуют уровням освоения роли по КРМ:

| Балл | Уровень | Характеристика (по КРМ) | 5-бальная шкала |
|---|---|---|---|
| 10 | Архитектор-визионер | Формирует стратегию, стандартизирует практики роли, признаки научной новизны или высокой практической ценности | Отлично |
| 9 | Продвинутый архитектор | Проектирует комплексные решения, обосновывает выбор архитектурных паттернов | Отлично |
| 9 | Проектировщик | Самостоятельно проектирует и оптимизирует компоненты, обосновывает нефункциональные требования | Отлично |
| 7 | Уверенный разработчик | Самостоятельно реализует роль на прикладной задаче, применяет лучшие практики | Хорошо |
| 6 | Самостоятельный | Самостоятельно реализует MVP по типовому паттерну | Хорошо |
| 5 | Начинающий | Реализует роль по инструкции, требует консультаций по нетиповым решениям | Удовлетворительно |
| 4 | Исполнитель | Применяет инструменты по инструкции, минимально приемлемый уровень для роли | Удовлетворительно |
| 3 | Ниже порога | Частично освоил инструменты, серьёзные пробелы | Неудовлетворительно |
| 2 | Фрагментарно | Демонстрирует лишь отдельные элементы роли | Неудовлетворительно |
| 0-1 | Не сформирован |Компетенция не подтверждена оценочным средством | Неудовлетворительно |

## Весовые коэффициенты оценочных средств
| Код |	Оценочное средство | Вес в итоговой оценке|
|---|---|---|
| ОС-1 | Лабораторные работы (3 работы) |	20% |
| ОС-2 | Практические задачи (типовые расчёты)/контрольные работы |	15% |
| ОС-3 | Коллоквиум (устный опрос) |	10% |
| ОС-4 | Тест (промежуточный) |	15% |
| ОС-5 | Зачёт (собеседование + практическая задача) |	40% |
